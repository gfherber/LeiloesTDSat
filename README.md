# Sistema de Casa de Leilões

## Sobre o projeto
Este é um sistema informatizado desenvolvido para gerenciar as rotinas de uma casa de leilões. O objetivo é controlar cadastros, lances e arremates. Este repositório foi criado para aplicar boas práticas de versionamento, segurança e qualidade de software exigidas pela empresa.

## Tecnologias utilizadas
- Java
- MySQL
